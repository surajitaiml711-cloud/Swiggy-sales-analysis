# 📌 Project Overview

This project performs an **Exploratory Data Analysis (EDA)** on **Swiggy Sales Data** using **Python, Pandas, NumPy, and Plotly**.

The objective is to uncover customer ordering patterns, revenue distribution, weekly and monthly sales trends, food preferences, and geographical performance through interactive visualizations.

---

# 🎯 Project Objectives

- Analyze monthly sales trends
- Analyze weekly sales performance
- Compare Veg vs Non-Veg revenue
- Identify top-performing states
- Analyze city-wise sales
- Understand daily ordering patterns
- Create interactive business dashboards using Plotly

# 📈 Key Insights

### ✅ Karnataka generated the highest revenue among all states.

### ✅ Bengaluru emerged as the highest revenue-generating city.

### ✅ Veg food contributed approximately **65%** of total revenue.

### ✅ Non-Veg contributed approximately **35%** of total revenue.

### ✅ Sales remained relatively stable across most weeks, indicating consistent customer demand.

### ✅ Weekend sales showed slightly higher revenue than weekdays.

### ✅ Monthly sales exhibited consistent performance with slight seasonal fluctuations.

# 🔍 Data Cleaning & Feature Engineering

The following preprocessing steps were performed:

- Converted Order Date into datetime format
- Removed missing values
- Created Month column
- Created Week column
- Extracted Day Name
- Classified dishes into Veg and Non-Veg categories
- Grouped data using Pandas GroupBy
- Created aggregated datasets for visualization
